# unityRVTTerrain
a runtime virtural texture terrain for unity 5.6
大地形的一种简化RVT
<b>因为 不同unity版本 地形shader差异较大,高版本不能直接运行出效果,需要先用unity5.6跑出截图效果,然后改你需要的版本里地形shader</b>
详细效果与思路 代码讲解 ：https://zhuanlan.zhihu.com/p/552748937
